# Buddy's Treehouse V5

An educational app for young learners featuring 42 interactive activities across 10 learning modules.

## Features

- 42 educational activities across 10 learning modules
- Custom Suno jingle integration
- Voice recorder (Talking Tom feature)
- Parent dashboard with analytics
- Gamification system (164 stars, 12 achievements)
- Professional animations and celebrations

## Tech Stack

- React 18.2 + TypeScript 5.3
- Vite 5.1 (build tool)
- Tailwind CSS 3.4
- Framer Motion 11.0 (animations)
- Zustand 4.5 (state management)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Learning Modules

1. Logic & Thinking - Puzzles and problem-solving
2. Science Explorer - Discover how the world works
3. Math Mountain - Climb to the top with numbers
4. Reading Rainbow - Adventures in letters and words
5. Art Studio - Express yourself through art
6. Music Meadow - Make music and learn rhythm
7. Friendship Forest - Learn about feelings
8. Active Adventures - Move your body
9. Nature Nook - Explore the natural world
10. Code Canyon - Introduction to coding concepts
